Как добавить кастомные стикеры:

1. Открыть любую комнату
2. Выбрать Настройки комнаты
3. Выбрать Подробнее
4. Открыть Инструменты разработчика
5. Выбрать Отправка данных учетной записи 
6. В Тип события ввести m.widget
7. Содержимое события вставить джейсон:

{
    "stickerpicker": {
        "content": {
            "type": "m.stickerpicker",
            "url": "https://bober80.github.io/stickerpicker/oz/?theme=$theme",
            "name": "Stickerpicker",
            "data": {}
        },
        "sender": "Your name",
        "state_key": "stickerpicker",
        "type": "m.widget",
        "id": "stickerpicker"
    }
}

8. Нажать Отправить
9. Закрыть настройки и нажать кнопку отправки стикера, должен загрузиться виджет со стикерами
10. Если не пояится, то закрыть/открыть клиента
